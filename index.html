About us

Information of our project:

We are a team of five students from ImiX flow of Kyiv Politechnic Institute. The theme of this site is a web service for creating block diagrams of code in the C programming language. You can also download your flowchart as an image and add it to your projects. The source code is publicly available in the GitHub repository.

Code Requirements!

Your code must be written correctly, follow the syntax rules! Also it must consist of a single "main" function, any other external functions will not be processed!

Our project is under development, so some of your programs may not be processed correctly. If you have a problem, please contact us!
